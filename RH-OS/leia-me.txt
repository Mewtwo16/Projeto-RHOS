No diretorio para conseguir trabalhar no projeto execute: 


- Criar package.json
npm init -y

- Instalar typescript
npm install typescript --save-dev

- Criar arquivo de configuração typescript
npx tsc --init

- instalar electron
npm install electron --save-dev

- instalar knex
npm install knex 

- instalar mysql2
npm install mysql2

- Instalar dotenv
npm install dotenv

###################
Essencial
###################

Criar arquivo .env e adicionar no .gitignore

modelo de configuração:

DB_HOST=localhost
DB_PORT=3306
DB_USER=root
DB_PASSWORD=sua senha
DB_DATABASE=RHOS
